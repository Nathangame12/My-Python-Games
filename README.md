# 🐍 My Python Games

Este repositório contém uma coleção de jogos e ferramentas desenvolvidos em Python, focados em lógica de programação, estruturas de dados e interação com o usuário via terminal.

  This repository contains a collection of games and tools developed in Python, focusing on programming logic, data structures, and terminal-based user interaction.

  ---

  ## 🎓 Academic Context
  Most of the projects listed here were developed as part of the practical curriculum for the **Saquarema Programming and Entrepreneurship Course (EPES)**. This course was the foundation for consolidating my knowledge in logic and software development.

  > **Special Note:** The **RPG** project was developed completely independently. I used the fundamentals learned at EPES to create a more complex gaming experience, featuring custom decision systems and narratives.

  ---

  ## 🎮 Games Included

  ### 🛡️ RPG (Independent Project)
  A text-based adventure game using attribute and choice systems. It is my most complex project so far, where I applied advanced logic to create a personalized journey for the player.

  ### 😵 Hangman Game (Jogo da Forca)
  A classic recreated in Python. It involves string manipulation, lists, and loops to validate player attempts.

  ### ✊ Rock, Paper, Scissors (Pedra, Papel e Tesoura)
  Implementation of "Jokenpô" using the `random` library to generate computer moves and conditional structures to define the winner.

  ### 🔢 Guessing Game (Jogo de Adivinhação)
  My first Python project! A logic challenge where the player must guess the secret number generated by the machine within a limited number of attempts.

## 🎮 Jogos Incluídos

### 🛡️ RPG (Projeto Independente)
Um jogo de aventura em texto que utiliza sistemas de atributos e escolhas. É o meu projeto mais complexo até agora, onde apliquei lógica avançada para criar uma jornada personalizada para o jogador.

### 😵 Jogo da Forca
Um clássico recriado em Python. Trabalha com manipulação de strings, listas e estruturas de repetição para validar as tentativas do jogador.

### ✊ Pedra, Papel e Tesoura
Implementação do famoso "Jokenpô" utilizando a biblioteca `random` para gerar jogadas do computador e estruturas condicionais para definir o vencedor.

### 🔢 Jogo de Adivinhação
Meu primeiro projeto em Python! Um desafio de lógica onde o jogador precisa descobrir o número secreto gerado pela máquina em um número limitado de tentativas.
