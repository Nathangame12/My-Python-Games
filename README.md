# 🐍 My Python Games

Este repositório contém uma coleção de jogos e ferramentas desenvolvidos em Python, focados em lógica de programação, estruturas de dados e interação com o usuário via terminal.

---

## 🎓 Contexto Acadêmico
A maioria dos projetos listados aqui foram desenvolvidos como parte do cronograma prático do **Curso de Programação e Empreendedorismo de Saquarema (EPES)**. O curso foi a base para consolidar meus conhecimentos em lógica e desenvolvimento de software.

> **Nota Especial:** O projeto de **RPG** foi desenvolvido de forma totalmente independente. Utilizei os fundamentos aprendidos no EPES para criar uma experiência de jogo mais complexa, com sistemas de decisão e narrativa próprios.

---

## 🎮 Jogos Incluídos

### 🛡️ RPG (Projeto Independente)
Um jogo de aventura em texto que utiliza sistemas de atributos e escolhas. É o meu projeto mais complexo até agora, onde apliquei lógica avançada para criar uma jornada personalizada para o jogador.

### 😵 Jogo da Forca
Um clássico recriado em Python. Trabalha com manipulação de strings, listas e estruturas de repetição para validar as tentativas do jogador.

### ✊ Pedra, Papel e Tesoura
Implementação do famoso "Jokenpô" utilizando a biblioteca `random` para gerar jogadas do computador e estruturas condicionais para definir o vencedor.

### 🔢 Jogo de Adivinhação
Meu primeiro projeto em Python! Um desafio de lógica onde o jogador precisa descobrir o número secreto gerado pela máquina em um número limitado de tentativas.
